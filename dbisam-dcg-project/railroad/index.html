identifier (lexical leaf)

A DBISAM identifier: a bare word (letter/underscore then letters, digits, underscores), a "double-quoted" name, or a [bracketed] name (bracket contents must still be a valid bare identifier). Drawn as a single terminal; the three lexical forms are not expanded.

string_literal (lexical leaf)

A single-quoted string with SQL-standard doubled-quote escaping ('O''Brien'). Backslashes are literal.

integer_literal (lexical leaf)

A run of decimal digits.

decimal_literal (lexical leaf)

A decimal / float literal: 3.14, .5, 5., 1.5e3, 1E-5, etc.
